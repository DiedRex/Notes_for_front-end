#Linux指令大全
1. mkdir box -------------- 在目录下创建一个名为box的文件夹
2. ls all,简写 ls -al ---------------- 列出目录下的所有文件(包括隐藏文件)
3. ls ----------------列出目录下的文件，不包括隐藏文件
4. cd box（box为文件夹）-----------------进入到box文件夹
5. cd ~ -----------------回到最大的主目录
6. cd .. ----------------放回上一级目录
7. vim 文件名 ----------修改编辑文件
	- 推出时，按esc，然后冒号:q，直接退出。:wq保存后退出
